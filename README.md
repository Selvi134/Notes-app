Notes App (React)

A simple and user-friendly Notes Application built with React.
This app allows users to add, edit, delete, and save notes with persistent storage using localStorage.
It is an excellent beginner-friendly project that demonstrates React hooks, state management, and basic UI design.

Features

-Add new notes

-Edit existing notes

-Delete notes

-Automatically saves notes using localStorage

-Input auto-focus using useRef

-Clean and minimal UI

Technologies Used

-React (useState, useRef, useEffect)

-JavaScript

-CSS (basic styling)

-localStorage API
